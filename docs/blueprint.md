# **App Name**: DataVisor

## Core Features:

- Dashboard Display: Display key metrics from the selected table data in a dashboard format (e.g., total meetings scheduled).
- Responsive Table: Present tabular data in a responsive, easy-to-navigate table format.
- Selection Highlighting: Highlight and track which row(s) are selected.
- Real-time Updates: Update the dashboard metrics in real-time based on selected rows in the table.

## Style Guidelines:

- Primary color: Sky blue (#63B5FF) for a calm and trustworthy feel.
- Background color: Very light blue (#F0F8FF) to ensure readability and a clean aesthetic.
- Accent color: Electric purple (#BF40BF) to highlight interactive elements.
- Font: 'Inter', sans-serif, for both headings and body text due to its clean and readable design suitable for data-heavy applications.
- Dashboard metrics are displayed prominently at the top, with the responsive table below. Use clear section dividers.
- Simple, flat icons to represent different metrics on the dashboard (e.g., a calendar for meetings).
- Subtle transitions when data is updated or when rows are selected in the table.
